# servlet = Server + Applet

* 데이터 전송방식
```
get : 데이터 조회 요청
post : 데이터 생성 요청
put : 데이터 수정 요청
delete : 데이터 삭제 요청
```

* Get 방식 (Default)
```
1. URL에 데이터를 포함시켜 요청 (Query string 방식)
2. 데이터를 패킷의 Header에 포함하여 전송
3. 전송하는 데이터의 길이에 한계 (1024 Byte)
4. 캐싱 가능함 (보안에 취약함)
```

* Post 방식
```
1. Url에 데이터를 노출하지 않고 요청
2. 데이터를 패킷의 Body에 담아 전송
3. 전송하는 데이터의 길이에 제한이 없음
4. 캐싱할 수 없음 (보안에 강함)
```
